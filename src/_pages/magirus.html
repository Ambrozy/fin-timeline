Задача от заказчика

Создать корпоративный сайт представительства Magirus в России.
Разработали дизайн webboxstd.com.

Мое решение в коде

Особенность сайта: многоуровневый каталог товаров с максимальной вложенностью 4. Для каждого пункта меню требовалось подготовить и загрузить свою картинку. Последующее редактирование и замена картинок выведена в CMS.
Нестандартный вывод карточки товара по блокам: описание, исполнение, характеристики, буклет. В каждом блоке, где требуется вводить текст в CMS был использован редактор TinyMCE.

Результат

Верстка по готовому дизайну. Посадка на CMS ModX. Для каждого вида страниц в CMS создан отдельный шаблон редактирования.
Перейти на сайт: http://www.magirus.ru/
