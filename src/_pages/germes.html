Задача от заказчика

Создать корпоративный сайт по готовому дизайну. Дизайн предоставил сам заказчик.

Мое решение в коде

Особенность сайта: много страниц с различным выводом, многоуровневый каталог, у каждого товара своя форма заказа. В CMS все страницы были сгруппированы по смыслу для удобной навигации. Поля текстового ввода используют редактор TinyMCE. Настройки сайта выведены в отдельное меню.

Результат

Верстка по готовому дизайну. Посадка на CMS ModX. Для каждого вида страниц в CMS создан отдельный шаблон редактирования. Страницы сгруппированы по смыслу.
